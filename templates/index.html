CARDIOVISION

About Cardio Vision




The project is a machine learning-based system designed to detect and diagnose retinal diseases using fundus images. It aims to provide an efficient and accurate solution for early diagnosis, which can potentially save many lives by preventing irreversible damage to the eyes. By leveraging the power of deep learning and computer vision, this project has the potential to revolutionize the field of ophthalmology and make eye care more accessible to everyone.

Our Services

Automated detection of eye diseases

Our project uses a machine learning model to accurately detect various eye diseases such as Dry AMD, Wet AMD, Mild/Moderate/Severe/Proliferate DR, Cataract, Pathological myopia, and Glaucoma.

Early diagnosis and prevention

Our model allows for early diagnosis and prevention of eye diseases, helping patients receive timely treatment and potentially avoiding further complications.

Efficient and accessible healthcare

With our automated detection system, healthcare professionals can efficiently screen patients for eye diseases, making healthcare more accessible to those who need it.

Know More

The Machine Learning Model

The machine learning model used in this project is a convolutional neural network (CNN) based classification model that can classify retinal fundus images . It was built using the TensorFlow and Keras libraries and trained on a subset of the dataset using a combination of data augmentation and transfer learning techniques.

The Dataset

The dataset used in this project consists of retinal fundus images of patients with various eye diseases such as age-related macular degeneration (AMD), diabetic retinopathy (DR), cataract, pathological myopia, and glaucoma. The images were acquired from different sources and preprocessed for uniformity.
